# EPIC: Board MVP

## Objetivo

Construir una herramienta visual que permita organizar habitaciones de un retiro con 30–100+ personas sin usar Excel.

## Alcance

Incluye:

- Board visual
- Gestión de participantes
- Gestión de grupos
- Pendientes agregados
- Pre-asignación básica
- Undo

No incluye:

- Marketplace
- Pagos
- Simulación avanzada
- Roles
- Log visible

## Definition of Done

- Organizador puede:
  - Crear evento
  - Añadir 60 participantes
  - Crear grupos
  - Asignar habitaciones manualmente
  - Pre-asignar no asignados
  - Resolver conflictos
  - Descartar dudosos
  - Exportar plan
- Todo sin usar Excel.
