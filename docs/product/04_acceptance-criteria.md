# Acceptance Criteria (MVP)

## Board

- Genera habitaciones automáticamente al crear evento.
- Permite renombrar habitaciones.
- Permite cerrar habitación.
- Permite exceder capacidad con conflicto visible.

## Participantes

- Permite añadir lista pegada.
- Muestra nombre compacto.
- Permite editar panel lateral.
- Permite cambiar estado.
- Permite descartar y restaurar.
- No elimina datos al descartar.

## Grupos

- Permite crear grupo.
- Permite marcar strong/flexible.
- Permite dividir grupo.
- Permite congelar grupo.

## Pendientes

- Contador único visible.
- Incluye dudosos.
- Incluye alergias.
- Incluye conflictos.
- Incluye preferencias no resueltas.

## Pre-asignación

- Solo afecta no asignados.
- No altera asignaciones existentes.
- Se puede deshacer con un solo Undo.

## Undo

- Visible.
- Revierte acción completa.
